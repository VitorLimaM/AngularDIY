# Angular Academic Project 

This project was developed as part of my Front-End Development college program to practice Angular fundamentals, including components, routing, forms, and observables.

## 📌 Project Overview

The goal of this application is to apply core Angular concepts learned during coursework through practical implementation. This project focuses on building a functional and interactive web application using Angular best practices.

## 🚀 Features

- Angular Components
- Data Binding (One-way & Two-way)
- Event Handling
- Angular Routing
- Reactive Forms
- Form Validation
- Observables (RxJS)
- Dynamic UI updates
- Responsive layout

## 🛠️ Technologies Used

- Angular
- TypeScript
- RxJS
- HTML5
- CSS3
- Node.js
- Angular CLI

## 📦 Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/project-name.git
```

2. Navigate into the project folder:

```bash
cd project-name
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
ng serve
```

5. Open your browser and go to:

```
http://localhost:4200
```
